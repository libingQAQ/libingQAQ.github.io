初学Java-Arrays

一、定义

Arrays类位于 java.util 包中，是一个工具类，不能实例化，主要包含了操纵数组的各种方法(搜索、排序、复制、比较).

二、常用方法

1.排序

sort方法可以对数组进行排序，并且提供多种重载形式，用于满足各种类型数组的排序需要

基本数据类型

sort(int[] a)

对基本数据类型数组的所有元素进行升序排序。

int a[] = {9,5,6,13,3,8};
Arrays.sort(a);
System.out.println(Arrays.toString(a));
//[3, 5, 6, 8, 9, 13]

sort(int[] a, int formIndex, int toIndex)

对基本数据类型数组[formIndex, toIndex)范围内的元素进行升序排序

int a[] = {9,5,6,13,3,8};
Arrays.sort(a, 1 , 5);
System.out.println(Arrays.toString(a));
//[9, 3, 5, 6, 13, 8]

引用类型

sort(Object[] a)

对引用类型数组的元素进行排序，要求数组元素都实现了Comparable接口

Integer a[] = {9,5,6,13,3,8};

//Integer实现了Comparable接口
Arrays.sort(a);
System.out.println(Arrays.toString(a));
//[3, 5, 6, 8, 9, 13]

sort(Object[] a, int formIndex, int toIndex)

对引用类型数组[formIndex, toIndex)范围内的元素进行排序，要求数组元素实现了Comparable接口

Integer a[] = {9,5,6,13,3,8};
//Integer实现了Comparable接口

Arrays.sort(a, 1, 5);
System.out.println(Arrays.toString(a));
//[9, 3, 5, 6, 13, 8]

sort(T[] a, Comparator<? super T> c)

对引用类型数组的元素进行排序，需要实现一个Comparator接口 (定义排序条件)

Integer a[] = {9,5,6,13,3,8};
rrays.sort(a, (o1, o2) -> {return o2.compareTo(o1);} ); 
//可以用lambad表达式 简化代码，这里是降序排序

System.out.println(Arrays.toString(a));
//[13, 9, 8, 6, 5, 3]

void sort(T[] a, int fromIndex, int toIndex, Comparator<? super T> c)

对引用类型数组[formIndex, toIndex)范围内的元素进行排序，需要实现一个Comparator接口 (定义排序条件)

Integer a[] = {9,5,6,13,3,8};
Arrays.sort(a, 1, 5, (o1, o2) -> {return o2.compareTo(o1);});
//可以用lambad表达式 简化代码，这里是降序排序

System.out.println(Arrays.toString(a));
//[9, 13, 6, 5, 3, 8]

2.搜索

binarySearch方法用来搜索指定的数组，以获得指定对象，提供多种重载形式，用于满足各种类型数组的查找需要。搜索之前，需要对数组使用sort方法进行排序。

基本数据类型

int binarySearch(int[] a, int key)

在基本数据类型数组中搜索key，返回

  1. 存在key，则返回任意一个key值的下标
  2. 不存在key, 则返回 [-(插入点) - 1] 读作 [负插入点减一] (插入点：将key插入到合适位置的下标)
int a[] = {1,3,4,5,5,8,9};
int bs10 = Arrays.binarySearch(a, 10);
int bs6 = Arrays.binarySearch(a, 6);
int bs5 = Arrays.binarySearch(a, 5);
int bs0 = Arrays.binarySearch(a, 0);

System.out.println("binarySearch: 10 = " + bs10);
// binarySearch: 10 = -8  10应该插入到 7，-7 - 1 = -8

System.out.println("binarySearch: 6 = " + bs6);
//binarySearch: 6 = -6    6应该插入到 5， -5 - 1 = -6

System.out.println("binarySearch: 5 = " + bs5);
//binarySearch: 5 = 3   

System.out.println("binarySearch: 0 = " + bs0);
//binarySearch: 0 = -1    0应该插入到 0， -0 - 1 = -1

int binarySearch(int[] a, int fromIndex, int toIndex, int key)

在基本数据类型数组[formIndex, toIndex)范围内搜索key，返回

  1. 存在key，则返回任意一个key值的下标
  2. 不存在key, 则返回 [-(插入点) - 1] 读作 [负插入点减一] (插入点：将key插入到合适位置的下标)

注意：我们只在 [formIndex, toIndex) 中搜索，所有的插入点只能在 [formIndex, toIndex]中

int a[] = {1,3,4,5,5,8,9};
int bs10 = Arrays.binarySearch(a,1,5, 10);
int bs6 = Arrays.binarySearch(a,1,5,6);
int bs5 = Arrays.binarySearch(a,1,5, 5);
int bs0 = Arrays.binarySearch(a,1,5, 0);

System.out.println("binarySearch: 10 = " + bs10);
// binarySearch: 10 = -6 只能将10插入到 5， -5 - 1 = -6

System.out.println("binarySearch: 6 = " + bs6);
// binarySearch: 6 = -6  只能将6 插入到 5， -5 - 1 = -6

System.out.println("binarySearch: 5 = " + bs5);
// binarySearch: 5 = 3

System.out.println("binarySearch: 0 = " + bs0)
// binarySearch: 0 = -2 只能将0 插入到 1， -1 - 1 = -2

引用类型

int binarySearch(Object[] a, Object key)

在引用类型数组里搜索key，要求数组元素实现Comparable接口，返回值参照int binarySearch(int[] a, int key) 方法

Integer a[] = {1,3,4,5,5,8,9};
//Integer自己实现了Comparable接口

int bs6 = Arrays.binarySearch(a, 6);
int bs5 = Arrays.binarySearch(a, 5);
int bs0 = Arrays.binarySearch(a, 0);

System.out.println("binarySearch: 10 = " + bs10);
// binarySearch: 10 = -8  10应该插入到 7，-7 - 1 = -8

System.out.println("binarySearch: 6 = " + bs6);
//binarySearch: 6 = -6    6应该插入到 5， -5 - 1 = -6

System.out.println("binarySearch: 5 = " + bs5);
//binarySearch: 5 = 3   

System.out.println("binarySearch: 0 = " + bs0);
//binarySearch: 0 = -1    0应该插入到 0， -0 - 1 = -1

int binarySearch(Object[] a, int fromIndex, int toIndex, Object key)

在引用类型数组[formIndex, toIndex)范围内搜索key，要求数组元素实现Comparable接口，返回值参照int binarySearch(int[] a, int fromIndex, int toIndex, int key)方法

Integer a[] = {1,3,4,5,5,8,9};
//Integer自己实现了Comparable接口

int bs10 = Arrays.binarySearch(a,1,5, 10);
int bs6 = Arrays.binarySearch(a,1,5,6);
int bs5 = Arrays.binarySearch(a,1,5, 5);
int bs0 = Arrays.binarySearch(a,1,5, 0);

System.out.println("binarySearch: 10 = " + bs10);
// binarySearch: 10 = -6 只能将10插入到 5， -5 - 1 = -6

System.out.println("binarySearch: 6 = " + bs6);
// binarySearch: 6 = -6  只能将6 插入到 5， -5 - 1 = -6

System.out.println("binarySearch: 5 = " + bs5);
// binarySearch: 5 = 3

System.out.println("binarySearch: 0 = " + bs0)
// binarySearch: 0 = -2 只能将0 插入到 1， -1 - 1 = -2

3.填充

fill方法是填充数组，提供多种重载形式，用于满足各种类型数组的填充需要。

基本数据类型

void fill(int[] a, int val)

在基本数据类型数组中填充指定值val。

int a[] = new int[6];
Arrays.fill(a,3);
System.out.println(Arrays.toString(a));
//[3, 3, 3, 3, 3, 3]

void fill(int[] a, int fromIndex, int toIndex, int val)

在基本数据类型数组[formIndex, toIndex)范围内中填充指定值val。

int a[] = new int[6];
Arrays.fill(a, 2, 5, 3);
System.out.println(Arrays.toString(a));
//[0, 0, 3, 3, 3, 0]

引用数据类型

void fill(Object[] a, Object val)

在引用类型数组中填充指定对象val。

Integer a[] = new Integer[6];
Arrays.fill(a, new Integer(3));
System.out.println(Arrays.toString(a));
//[3, 3, 3, 3, 3, 3]

void fill(Object[] a, int fromIndex, int toIndex, Object val)

在**引用类型数组[formIndex, toIndex)范围内**中填充指定对象val。```java
Integer a[] = new Integer[6];
Arrays.fill(a, 2, 5, new Integer(3));
System.out.println(Arrays.toString(a));
//[0, 0, 3, 3, 3, 0]

4.复制

copyOf方法和copyOfRange方法可以拷贝一个数组副本，提供多种重载形式，用于满足各种类型数组的复制需要。

基本数据类型

int[] copyOf(int[] original, int newLength)
返回一个长度为newLength的数组，

如果newLeng大于源数组的长度，多余的元素为0值;

如果newLeng小于源数组的长度，只拷贝源数组前面一部分的元素。

引用类型

T[] copyOf(T[] original, int newLength)

返回一个和源数组同一类型，长度为newLength的数组
