初学Java-Map

一、Map概述

Map是一种把键对象和值对象进行关联的容器。一个值对象又可以是一个Map，以此类推，这样就可以形成一个多级映射。Map容器中的键对象不允许为重复，具有唯一性。值对象则没有唯一性要求。

二、通用方法

下面的方法是所有实现Map接口的实现类都需要实现的方法。

新增方法

V put(K key, V value))

添加key，value到 容器中

void putAll(Map<? extends K, ? extends V> m)

将一个map容器中的所有元素都添加进来

查看方法

Set<Map.Entry<K,V>> entrySet()

返回map容器到一个Set集合中，以map集合中的Key=Value的形式返回到set中。

Set keySet()

返回map容器的所有key到一个Set集合中。

Collection values();

返回map容器的所有value到一个集合中。

V get(Object key)

根据map集合中元素的Key来获取相应元素的Value

boolean containsKey(Object key)

判断map容器中是否包含key

boolean containsValue(Object value)

判断map容器中是否包含value

boolean isEmpty()

判断map容器是否为空

int size()

删除方法

V remove(Object key)

void clear()

实现类

HashMap

HashMap的底层是 数组 + 链表 / 红黑树

/**
* The table, initialized on first use, and resized as
* necessary. When allocated, length is always a power of two.
* (We also tolerate length zero in some operations to allow
* bootstrapping mechanics that are currently not needed.)
*/
transient Node<K,V>[] table;

这是数组

/**
* Basic hash bin node, used for most entries.  (See below for
* TreeNode subclass, and in LinkedHashMap for its Entry subclass.)
*/
static class Node<K,V> implements Map.Entry<K,V> {
    final int hash;
    final K key;
    V value;
    Node<K,V> next;
}

这是链表

/**
* Entry for Tree bins. Extends LinkedHashMap.Entry (which in turn
* extends Node) so can be used as extension of either regular or
* linked node.
*/
static final class TreeNode<K,V> extends LinkedHashMap.Entry<K,V> {
    TreeNode<K,V> parent;  // red-black tree links
    TreeNode<K,V> left;
    TreeNode<K,V> right;
    TreeNode<K,V> prev;    // needed to unlink next upon deletion
    boolean red;
}

这是红黑树

当数组元素不超过64或链表内节点不超过8时，是 数组 + 链表，否则 链表 会升级成 红黑树

适用于在Map中插入、删除和定位元素。但是遍历键是无序的(如果出现有序的情况，那是你数据量太小了)。

Map<Integer,Integer> map = new HashMap<>();

System.out.print("添加：");
for (int i = 0; i < 10; i++) {
    int key =(int) (Math.random() * 100);
    if(!map.containsKey(key)){
        map.put(key,key);
        System.out.print(key + "=" + key + " ");
    }
}

System.out.println();
System.out.print("查看：");
map.entrySet().forEach(a->{System.out.print(a + " ");});

输出结果：

添加：46=46 38=38 98=98 57=57 30=30 13=13 40=40 3=3 6=6 62=62 
查看：98=98 3=3 38=38 6=6 40=40 57=57 13=13 46=46 30=30 62=62

TreeMap

TreeMap的底层是红黑树

private transient Entry<K,V> root;
/**
    * Node in the Tree.  Doubles as a means to pass key-value pairs back to
    * user (see Map.Entry).
    */

static final class Entry<K,V> implements Map.Entry<K,V> {
    K key;
    V value;
    Entry<K,V> left;
    Entry<K,V> right;
    Entry<K,V> parent;
    boolean color = BLACK;
}

TreeMap的底层是红黑树所有性能稍逊HashMap，但是它的遍历是有序的，是以key值排序参照。适用于按自然顺序或自定义顺序遍历键(key)

Map<Integer,Integer> map = new TreeMap<>();
Random random = new Random();

for (int i = 0; i < 10; i++) {
    int key = random.nextInt(100);
    int value = random.nextInt(100);
    map.put(key,value);
}

System.out.print("key  ：");
map.keySet().forEach(a->{System.out.printf("%3d",a);});

System.out.print("\nvalue：");
map.values().forEach(a->{System.out.printf("%3d",a);});

输出结果

key  ：  9 25 27 37 44 54 61 79 91 93   //以key为yi排序
value： 26 82 23 62 94  3  2 16 10 75
